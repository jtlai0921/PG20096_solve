練習問題2-1（原問題2-3）
請寫出一個程式使其輸出結果為左方整數是右方整數的百分比(如右所示)。

顯示結果(for問題)
請輸入兩個整數：
整數A：54
整數B：84
A是B的64%。

程式碼
/*
顯示讀取的兩個整數值中前者為後者的多少%
*/

#include <stdio.h>

int main(void)

{
    int na, nb;
   
    puts("請輸入兩個整數：");
    printf("整數A：");    scanf("%d", &na);
    printf("整數B：");    scanf("%d", &nb);

    printf("A是B的%d。\n", 100 * na / nb);
    
    return (0);
}

執行範例
請輸入兩個整數：
整數A：54
整數B：84
A是B的64%。

練習問題2-4（原問題2-9）
請寫一個程式進行整數常數、浮點數常數、int型態變數和double型態變數的交叉相乘或相除，以確認前面各項規則。

程式碼
/*
　確認型態與運算
*/										

#include  <stdio.h>	

int main(void)	
{
    int        nx;		/* 整數 */
    double dy;		/* 浮點數 */

    nx = 5    /  2;		/* nx ← 2    */
    dy = 5.0 /  2.0;		/* dy ← 2.5 */

    printf("5    /  2    = %d\n",    5    /  2   );
    printf("5    /  2.0 = %f\n",     5    /  2.0);
    printf("5.0 * 2.0 = %f\n\n",  5.0 * 2.0);
	
    printf("5    /  nx  = %d\n",    5    /  nx);
    printf("5    /  dy  = %f\n",     5    /  dy);
    printf("5.0 /  nx  = %f\n",     5.0 /  nx);
    printf("5.0 /  dx  = %f\n",     5.0 /  dy);
    printf("nx  /  dy  = %f\n",     nx  /  dy);
    return (0);
}

執行結果
nx = 2
dy = 2.500000

5     / 2    = 2
5     / 2.0 = 2.500000
5.0 * 2.0 = 10.000000

5     / nx = 2
5     / dy = 2.000000
5.0  / nx = 2.500000
5.0  / dy = 2.000000  
nx   / dy = 0.800000

練習問題2-6（原問題2-12）
請寫一個程式使其輸入身高（整數）、輸出結果為標準體重（實數）。標準體重的公式是（身高－100）×0.9，算到小數點以下一位。

顯示結果(for 問題)
請輸入身高：175
您的標準重是67.5。

程式碼
/*  輸入身高再以實數顯示標準體重
*/

#include <stdio.h>

int main(void)
{
    int         height;
    double  weight;

    printf("請輸入身高：");
    scanf("%d, &height);

    weight = (height - 100) * 0.9;

    printf("您的標準重是%.lf。\n", weight);

    return (0);
}

顯示結果
請輸入身高：175
您的標準重是67.5。
